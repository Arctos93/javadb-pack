0.  Przydatne informacje:

    CRUD ​ (ang. ​ Create, ​ Read, ​ Update and ​ Delete) ​- cztery podstawowe funkcje:
        Create - INSERT, Read - SELECT, Update - UPDATE, Delete - DELETE

    DAO ​ (ang. ​ Data Access ​ Object) - ​komponent/klasa dostarczający jednolity interfejs
        do komunikacji między aplikacją a źródłem danych (np.: bazą danych)

    DTO ​ (ang. ​ Data ​ Transfer​ ​ Object) ​ - klasa przechowująca dane z tabelki,
    odwzorowanie jednego wiersza tabeli w kodzie programu Java
